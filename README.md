# Interview Process Simulator for Campus Hiring

A web-based application that simulates a real campus recruitment pipeline including aptitude, coding, and HR interview rounds, with candidate performance analytics dashboards.

## Tech Stack
- React.js
- Tailwind CSS
- Framer Motion
- Recharts
- LocalStorage

## Features
- Candidate registration
- Aptitude test simulation
- Coding round with timer
- HR behavioral round
- Final performance dashboard with charts
- Fully responsive UI
